USUARIOS

LISTAR TODOS

curl -X GET http://3.138.155.172:8080/api/usuarios

OBTENER POR ID
GET http://3.138.155.172:8080/api/usuarios/{id}
curl -X GET http://3.138.155.172:8080/api/usuarios/1

CREAR
POST http://3.138.155.172:8080/api/usuarios
Body (JSON): {"nombre":"Juan Perez","correo":"juan.perez@example.com"}
curl -X POST http://3.138.155.172:8080/api/usuarios \
  -H "Content-Type: application/json" \
  -d '{"nombre":"Juan Perez","correo":"juan.perez@example.com"}'

ACTUALIZAR
PUT http://3.138.155.172:8080/api/usuarios/{id}
Body (JSON): {"nombre":"Juan P.","correo":"juan.p@example.com"}
curl -X PUT http://3.138.155.172:8080/api/usuarios/1 \
  -H "Content-Type: application/json" \
  -d '{"nombre":"Juan P.","correo":"juan.p@example.com"}'

ELIMINAR
DELETE http://3.138.155.172:8080/api/usuarios/{id}
curl -X DELETE http://3.138.155.172:8080/api/usuarios/1
